NeuroNexus

Projet universitaire en groupe de 4

NeuroNexus est un logiciel de classification de données basé sur l'algorithme KNN (k-plus proches voisins). Il permet d'importer des fichiers CSV et de classifier dynamiquement les points en fonction de leurs attributs. Conçu avec Java et JavaFX, il offre une interface intuitive et des fonctionnalités avancées pour analyser et visualiser les données sous forme de graphes 2D et 3D.

Fonctionnalités

  • Importation des données via fichiers CSV
  • Classification KNN avec choix entre la distance Euclidienne et Manhattan
  • Sélection dynamique des axes et critères de classification
  • Évaluation de la robustesse du modèle via validation croisée
  • Visualisation interactive des points avec un graphe
  • Mode 3D pour une meilleure analyse des attributs

Technologies

Java, JavaFX, architecture MVC, OpenCSV

Compétences techniques acquises

Développement d'interface graphique avec JavaFX, Utilisation du pattern MVC, Implémentation de K-Nearest Neighbors (KNN, classification supervisée), Optimisation de la classification, Généricité des classes, Tests & Debugging

Savoir-être

Travail en équipe, Gestion de projet, Esprit analytique, Résolution de problèmes
